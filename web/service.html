Tecnologias implementadas

Tenemos las siguientes tecnologias que encontrarás en nuestro proyecto

Visual Studio Code


Entorno IDE via web donde los usuarios podrán instalar lenguajes de programación con compiladores para lenguajes como C/C++ o Java. Con el sistema de directorio simple de Linux

MySQL


Gestión integral de bases de datos en línea mediante PHPMyAdmin, permitiendo una administración completa y eficiente sin la necesidad de descargar ninguna aplicación adicional.Esta solución proporciona una plataforma accesible y conveniente para la gestión de datos.

S.O. Linux


Sistema operativo accesible a través de la web con capacidad de conexión mediante VNC, permitiendo un uso integral y sin restricciones de todos los recursos disponibles en el servidor. Esta solución asegura un acceso remoto eficiente y completo, optimizando el rendimiento y la funcionalidad sin limitaciones.

Las máquinas virtuales de Windows 11 permiten gestionar y probar aplicaciones, realizar prácticas de Pentesting y crear clientes para servicios. Ofrecen un entorno aislado y seguro para experimentar y desarrollar sin afectar el sistema principal. Ideales para desarrolladores y profesionales de seguridad informática.

RD Web Access permite a los usuarios acceder a escritorios y aplicaciones remotas a través de un navegador web. Ofrece acceso remoto seguro y se integra con Active Directory para la autenticación y autorización de usuarios. Facilita el trabajo remoto y la administración centralizada de recursos.
